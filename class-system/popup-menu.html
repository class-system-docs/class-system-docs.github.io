# 悬停菜单

悬停在屏幕右侧，点击展开/收起。

# 截图

点击唤起截图功能并打开截图界面。

操作 功能
Esc 退出
鼠标双击 退出
拖拽鼠标画出矩形 矩形截图

使用鼠标确定好截图区域后按下 确定 按钮即可完成截图，截图保存路径为 程序路径/screenshots ，同时贴图在桌面。

# 贴图

将图片固定在桌面。

TIP

贴图只会在完成截图后自动开启

单击贴图使其获取焦点后，有以下操作：

按键 功能
BackSpace 关闭贴图（不可再次开启）
Delete
鼠标中键
鼠标双击
鼠标滚轮 缩放贴图

# 表格

显示完整的课程表、抬饭表、值日表，表中当日列高亮。

鼠标点击上方选项卡以切换表格，在表格上滚动鼠标滚轮以调整字体大小。

# 帮助

用默认浏览器打开 ClassSystem 使用文档 。
